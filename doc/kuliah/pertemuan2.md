**Penjelasan**

1. Vulnerability adalah suatu kelemahan program atau system yang memungkinkan terjadinya eksploetasi system.
2.  Pada TCP/IP terdapat

1.
  1. Application
  2. Transport Layer
  3.  Internet Layer
  4. Network Access

1. MAC Address digunakan untuk mengidentifikasi seluruh perangkat jaringan secara unik agar dapat terhubung dan  untuk komunikasi jaringan local seperti user atau PC dengan Router yang digunakan
2. ARP  berfungsi untuk memetakan ipaddress menjadi MAC address.
3. Kelamahn DNS

1.
  1.
    1. User menjadi lebih terbatas
    2. Sulit untuk diimplementasikan.
    3. Nama DNS sangat terbatas (tidak dapat membuat banyak nama domain)

1. Cara mengecek MAC Address pada IP Address coding/code &quot;  **arp –a**&quot; pada commant promt kemudian Enter , setelah itu akan muncul IP Address beserta MAC Address yang digunakan
2. Mengetahui IP Address pada suatu domain seperti Web,. Ketik scrip&quot;  **nslookup namaweb**&quot; pada Command Prompt atau Terminal kemudian Enter, nanti akan muncul nama domain beserta IP Address yang digunakan
3. Untuk melakukan vulnerabilities pada Python

1. **a.** Install Python dan parameter – parameter lain yang diperlukan seperti Scapy dan lain – lain.
2. **b.**  Cek apakah Python sudah terinstall atau belum dengan coding &quot;  **python**&quot; dan Enter,
3. **c.**  muncul versi python maka python berhasil diinstal.
4. **d.**  Setelah selesai kita membuat script Python, kemudian jika selesai membuat scriptnya makan kita running atau jalankan dengan Command Prompt dengan cara

**Penutup**

**Kesimpulan**

Kesimpulan diatas bahwa vulnerabilities adalah celah dari sistem keamanan web yang memungkinkan bagi seorang hacker untuk menembus sistem keamanan dari web tersebut

**Saran**

Pembahasan vulnerabilities lebih dipahami dan lebih mengetahui cara menggunakannya, namun menggunakannya ke hal yang lebih baik dan tidak ada pihak lain yang dirugikan.
